# LogStream Project Roadmap

## Current State
- **LogStream.MAUI**: .NET MAUI app for cross-platform log streaming.

## Next Steps
1. **Core Library Development**
    - Create `LogStream.Core` for shared logic and functionality.
    - Refactor existing code to utilize the core library.

2. **Front-End Expansion**
    - Continue development and refinement of `LogStream.MAUI`.
    - Begin planning and prototyping `LogStream.Web` for desktop and web views.

3. **Release Plan**
    - Polish and finalize `LogStream.MAUI` for initial release.
    - After shipping `LogStream.MAUI`, focus on building and launching `LogStream.Web`.

## Future Goals
- Ensure seamless integration between all front-ends and the core library.
- Expand platform support and add new features based on user feedback.
- Maintain clear documentation and update roadmap as project evolves.
